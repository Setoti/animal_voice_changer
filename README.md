# 🍃 どうぶつ語ボイスチェンジャー 

アップロードした音声を、某スローライフゲームに登場する「どうぶつ語」風の音声にリアルタイムで変換するStreamlitアプリです。

## 🚀 アプリを試す
**[こちらをクリックして起動 (Streamlit Community Cloud)](https://animalvoicechanger.streamlit.app/)**

## 📝 概要
このアプリは、アップロードされた音声ファイルを独特の高速・高ピッチな「どうぶつ語」へと変換します。
単なるピッチ変更ではなく、ピッチを一定にするなどして、あの懐かしくも可愛らしい電子音を再現しました。

## ✨ 主な機能
- **話速調整・ピッチ調整**: 話すスピードや声の高さをスライダーでカスタマイズ可能。
- **音声ダウンロード**: 生成した音声をWAV形式で保存して、動画制作などの素材として利用可能。

## 🛠️ 技術的な仕組み
信号処理の観点から、以下のようなアプローチで実装しています：
- **ピッチシフト & タイムストレッチ**: サンプリング周波数の操作による独特の音色の再現。
- **StreamlitによるUI**: 複雑な音声処理バックエンドを、Pythonのみで直感的なWebインターフェースに統合。

## 🛠️ 使用技術
- **Language**: Python 3.12
- **Framework**: Streamlit
- **Libraries**: NumPy, Librosa, Pyworld
